# File Service

Клиент - серверное приложение для работы с файлмаи

    Функции клиента:    

    ПОЛУЧИТЬ список файлов
    ПОЛУЧИТЬ файл по его имени с сервера и сохранить на клиенте
    ОТПРАВИТЬ файл на сервер
    ОБНОВИТЬ файл на сервере
    УДАЛИТЬ файл с сервера

## Запуск сервера:
    make server

## Запуск клиента:
    make client
